# Grand Strategy — MVP

Браузерная стратегическая игра. MVP: Франция 1852 года.

## Структура проекта

```
grand-strategy/
├── index.html        — главная страница (весь интерфейс)
├── js/
│   ├── map.js        — карта мира, провинции, зум, перетаскивание
│   ├── game.js       — игровая логика (ходы, казна, ИИ-триггеры)
│   └── ui.js         — панели, попапы, уведомления, сущности на карте
├── portraits/        — (пусто) сюда добавлять портреты правителей
└── README.md
```

## Как запустить локально

Открой `index.html` в браузере. Карта загружается из CDN (нужен интернет).

## Как запустить через GitHub Pages (бесплатный хостинг)

1. Залей репозиторий на GitHub
2. Зайди в Settings → Pages
3. Source: выбери ветку `main`, папку `/ (root)`
4. Сохрани — через минуту игра будет доступна по адресу `https://ИМЯ.github.io/РЕПОЗИТОРИЙ`

## Что уже работает

- Карта мира (Natural Earth, public domain) с перетаскиванием и зумом
- Провинции Франции с тултипами
- Левая панель: правитель, правительство, парламент, данные страны
- Походовая система (кнопка «Следующий месяц»)
- Дипломатия и советник
